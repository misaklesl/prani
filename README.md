# 🎂 Narozeninové přání – Martine

Interaktivní narozeninové přání s animovanými dveřmi, konfety a přáním od celého týmu.

## Jak spustit

Stačí otevřít soubor `index.html` v prohlížeči – žádná instalace ani server není potřeba.

## Ovládání

| Klávesa | Akce |
|---------|------|
| `→` (šipka vpravo) | Otevře dveře |
| `←` (šipka vlevo) | Zavře dveře |
| Klik na dveře | Otevře / zavře |

## Struktura

```
birthday-card/
├── index.html   # hlavní stránka
├── style.css    # styly
├── script.js    # logika dveří a konfet
└── README.md
```

## Úpravy

- **Jméno / text přání** – edituj `index.html`, sekce `card-content`
- **Barva dveří** – změň hodnoty `#0d9e9e` / `#11b5b5` / `#0a7a7a` v `style.css`
- **Konfety** – pole `colors` v `script.js`
